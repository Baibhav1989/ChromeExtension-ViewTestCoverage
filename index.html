Privacy Policy - Apex Class Coverage Viewer

Effective date: April 17, 2026

Apex Class Coverage Viewer is a Chrome extension used to view Salesforce Apex code coverage and queue Apex test execution.

Data We Process

  • Salesforce session token (sid cookie) to authenticate Salesforce API requests.
  • Salesforce instance URL and API version.
  • Extension UI preferences (for example, method view and exclude package classes toggles).

How We Use Data

  • To call Salesforce Tooling API and display Apex coverage/test information requested by the user.
  • To open Salesforce Apex Test Queue pages when tests are queued.
  • To remember user settings for convenience.

Data Storage

Data is stored locally in Chrome extension storage on the user’s device.

Data Sharing

We do not sell or share personal data with third parties. Data is transmitted only to Salesforce endpoints required for extension functionality.

Retention

Data remains until the user clears extension storage or uninstalls the extension.
